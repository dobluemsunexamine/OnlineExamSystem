# 判断学号

## 1.接口描述

接口请求域名:http://www.bluemsun.work/register/checkStudentCode
注册时判断学号是否冲突
请求方式:get

## 2.输入参数

| 参数名称  | 必选  |  类型  |         描述         |
| :-------: | :---: | :----: | :------------------: |
|  studentCode  |  是   | String |        学号   |

## 3.输出参数

| 参数名称 |  类型  |        描述        |
| :------: | :----: | :----------------: |
| success | Number |  学号是否已存在  |
